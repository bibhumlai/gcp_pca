# GCP Professional Cloud Architect Study Guide

A comprehensive, interactive study guide for the Google Cloud Professional Cloud Architect (PCA) certification exam.

## 📚 Contents

- Core GCP Services with decision matrices (Compute, Storage, Networking, Security, AI/ML, Operations)
- Deep dives into all 4 official case studies (Cymbal Retail, Altostrat Media, EHR Healthcare, KnightMotives)
- Google Cloud Architecture Framework mapping
- Exam-style Q&A for every section
